#---Описание---

*CSV Аналитик* - это веб-приложение для загрузки, просмотра и анализа табличных данных в формате CSV. Приложение предоставляет интерактивный интерфейс для визуализации данных и статистического анализа без необходимости написания кода.

# Основные возможности

# Загрузка данных
- Поддержка CSV файлов с различными разделителями (`,`, `;`)
- Автоматическое определение кодировки (UTF-8, CP1251)
- Автоматическое распознавание типов данных (числа, даты, время)

# Визуализация данных
Приложение поддерживает **5 типов интерактивных графиков**:

1. Линейный/Точечный график - для анализа зависимостей между переменными
   - Выбор осей X и Y
   - Возможность группировки по цвету
   - Сортировка данных для линейных графиков

2. Пузырьковая диаграмма (Bubble Chart) - отображение 3-х переменных одновременно
   - Ось X (числовая)
   - Ось Y (числовая)
   - Размер пузырька (числовой) - визуализирует третий параметр
   - Цвет (числовой или категориальный) - четвертый параметр

3. Гистограмма распределения - анализ распределения данных
   - Настройка количества интервалов (bins)
   - Встроенный boxplot для дополнительной информации

4. Boxplot (ящик с усами) - визуализация статистических характеристик
   - Медиана, квартили, выбросы
   - Скрипичный график (Violin Plot) для детального анализа распределения

5. 3D график - визуализация трех числовых переменных
   - Интерактивное вращение
   - Цветовое кодирование

# Статистический анализ
- Основная статистика: среднее, медиана, стандартное отклонение
- **Дополнительная статистика: минимум, максимум, количество значений
- Квартили: 25%, 50%, 75%, 90%, 95%
- Корреляционная матрица для всех числовых переменных
- Анализ категориальных данных: частотные таблицы и круговые диаграммы

# Экспорт данных
- Скачивание любого графика в интерактивном HTML формате
- Сохранение полной функциональности (масштабирование, наведение)
